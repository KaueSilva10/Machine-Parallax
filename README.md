# 🔢 Conversor de Bases Numéricas (Machine)

Um conversor interativo e visual desenvolvido com HTML, CSS e JavaScript, utilizando efeitos de **Parallax Scrolling** e conversão de bases matemáticas em tempo real.

## 💻 Sobre o Projeto
Esta "Machine" permite que o usuário insira um número decimal e visualize instantaneamente sua conversão para as bases:
* **Octal**
* **Hexadecimal**
* **Binário**

O diferencial deste projeto é a interface imersiva com camadas de montanhas e estrelas que se movem conforme o scroll do usuário.

---

## 🛠️ Tecnologias Utilizadas
* **HTML5** (Estrutura)
* **CSS3** (Layout Flexbox e Efeitos Parallax)
* **JavaScript** (Lógica de conversão e manipulação de DOM)
* **Google Fonts** (Tipografia Poppins)
